Medical Cost Analysis (Excel Project)

📊 Overview
This project analyzes medical insurance costs based on factors like age, BMI, smoking habits, and region.

🔧 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Slicers (for interactivity)

📈 Analysis Performed
- Basic statistical analysis (average, median, standard deviation)
- Gender-wise comparison of medical charges
- Impact of smoking on medical costs
- Dependents vs medical charges analysis
- Region-wise cost analysis
- Cost distribution using histogram
- BMI classification analysis
- Correlation between BMI and charges
- Combined analysis of smoking and region

📊 Dashboard Features
- Interactive charts
- Filters using slicers (gender, smoker, region, BMI)
- Real-time data updates based on selection

🎯 Key Insights
- Smoking significantly increases medical costs
- Higher BMI is associated with higher charges
- Regional differences exist in medical expenses
